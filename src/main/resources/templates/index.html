网页转PDF和图片

 2018年12月 Spring Boot

已用于计量院项目中的设备证书生成和导出。项目参考自https://github.com/petterobam/my-html2file，修正其不能在linux内正常使用的问题（原项目的LinuxPath并不正确）， 并使用Apache PdfBox将生成出来的PDF文件转化为PNG图片，以解决用wkhtmltoimage生成出来的图片过大的问题。 即通过转PDF省去一步图片压缩的任务，而PDF又正好是业务需要的。核心逻辑是使用字符串拼接代码，帮助用户拼接wkhtmltopdf需要的参数指令， 避免不必要的麻烦。

对"高校体育"App的伪造请求服务

 2018年10月 Spring Boot

《高校体育》是一款上海市教委或者更高的上级教育部门推广的，为敦促广大高校学生积极锻炼用的类似于KEEP的跑步App。 其后端实现存在诸多问题：使用HTTP，App端代码混淆薄弱，数据加密方式低级，不校验用户身份，没有校方管理后台，跑步记录校验薄弱等。 故可以进行接口的伪造访问，达到一键跑步的目的。

基于Spring Boot和MVP架构的特色河蟹大赛评比管理系统

 2018年10月 Spring Boot+Android

本项目计划参加2019年上海市大学生计算机应用能力大赛并和后端项目一起作为我的毕业论文主题。 前端采用MVP架构的Android App，开发流程规范，便捷，实现了组件、业务的可拓展和良好的层次结构； 使用时下官方最新的Material Design的多个组件以及OkHttp、Glide等主流Android框架实现快速高效开发。 后端采用RESTFUL API风格的Spring Boot，使用JWT和AES加密限制请求的恶意伪造，防止用户跨用户组操作；全部Api都接入swagger，方便测试与维护； 接入网易云短信服务实现用户登录和注册的手机校验； 接入七牛云对象存储实现App的图文并茂。

基于安卓的定制化数字课堂考勤考评服务系统

 2017年12月 Android+Servlet

本项目是我自学Android后开发的第一款App。技术简陋，前后端代码冗余程度高，不易维护，故在2018年5月以后就停止了更新。 项目为解决本科课堂规模较大而导致的教师与学生不够熟悉的问题， 教师能够一键了解当前课堂出勤学生的数量和详细信息， 能够根据教室座位图对出勤的学生进行快速评价，能够方便地维护所有的课堂记录。 项目荣获2018年上海市大学生计算机应用能力大赛（程序设计应用组）上海市二等奖；2018年中国大学生计算机设计大赛（软件应用与开发类）全国三等奖。
